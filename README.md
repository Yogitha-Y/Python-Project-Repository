# 🐍 Python Fundamentals Project (Data Technician Bootcamp)

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp (Week 6 – Python)** and focuses on building foundational programming skills using Python.

The project includes a series of exercises and mini-programs designed to develop core coding concepts, including **user interaction, logical decision-making, and iterative processing**, with real-world inspired tasks such as calculators, number checks, and an ATM simulation.
